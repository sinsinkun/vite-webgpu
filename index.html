WebGPU Quickdraw


Description

WebGPU Quickdraw dramatically simplifies the steps required to setup a basic render pipeline using WebGPU, and configuring render objects to draw onto the canvas. Common vertex buffers are exposed, and MVP transforms are pre-built/automatically calculated by calling a short-hand function to update the render objects' position, rotation, and scale.


Note: WebGPU is not supported on Safari browsers as of this writing, and needs to be explicitly enabled on firefox. Please refer to the logs below for details.
